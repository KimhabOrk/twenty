## Usage

This guide will help you transition from a version of twenty to another.

## Upgrade from v0.21.0 to v0.22.0

Run the following:
```
yarn database:migrate:prod 
```
```
yarn command:prod workspace:sync-metadata -f 
```
```
yarn command:prod upgrade-0.22
```
